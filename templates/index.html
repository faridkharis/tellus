Tell - Us

Tell-Us merupakan aplikasi berbasis web yang menerapkan Algoritma K-Means Clustering untuk melakukan text xlustering. Text Clustering merupakan pengelompokkan teks ke dalam beberapa kelompok berdasarkan kemiripan teks tersebut. Teks yang menjadi objek aplikasi ini adalah teks ulasan aplikasi startup yang diperoleh dari website Google Playstore.

Tahapan penggunaan Tell-Us adalah sebagai berikut :

1. Upload file yang berisi ulasan dari Google Play dengan format .csv
2. Upload file stopword
3. Upload file normalisasi
4. Tekan tombol "Klasterisasi"
